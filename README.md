#  Susi Air Crew Portal

A modern and responsive Crew Portal web application built with **Nuxt 4.5.0**, **Vue 3.5.40**, **TypeScript**, and **Pinia**.

This project is a frontend implementation of a pilot dashboard that helps crew members monitor flight hours, upcoming flights, schedules, and document through a responsive interface.

#  Features

### Authentication

- Mock Login
- Protected Routes using Middleware
- Guest Middleware
- Logout

### Dashboard

- Pilot Overview
- Flight Hours Summary
- Annual Hours Limit
- Rolling Flight Hours
- Flight Hours Chart
- Upcoming Flight
- Latest News
- Responsive Hero Section

### Documents

- Certificate & License Overview
- Expiration Status
- Days Remaining Indicator

### Schedule

- Monthly Calendar View
- Duty Type Color Legend
- Current Date Highlight

### User Experience

- Responsive Layout
- Sidebar Navigation
- Mobile Bottom Navigation
- Skeleton Loading
- Smooth Page Transition
- Reusable Components

#  Getting Started

Clone this repository

```bash
git clone https://github.com/nurfaizah/susiair-crew-portal.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
